О приложении

Это маленькое, приложение для прослушивания онлайн радио и музыки, сделано в рамках моего обучения CSS и JavaScript технологиям.

Приложение содержит мои эксперименты и любимые мной ссылки на аудио/видео потоки взятые из открытых источников в сети интернет и только проигрывает их посредством JS.

Вы можете скачать приложение для android OS, установить его на телефон, планшет, автомагнитолу или домашний медиаплеер. Теперь ГлавРадио будет всегда в паре кликов от вас!
